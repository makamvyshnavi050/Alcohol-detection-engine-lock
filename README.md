# Alcohol-detection-engine-lock

# Project Description
Drunk driving is a major contributor to road accidents worldwide. This project proposes a cost-effective and efficient solution to prevent drunk driving using an Arduino-based smart engine lock system. The system integrates an alcohol sensor to analyze the breath of the driver. The sensor detects alcohol content in the driver's breath and sends the data to the Arduino microcontroller. If the alcohol level exceeds the predetermined threshold, the engine is automatically locked, preventing the vehicle from starting. The proposed system promotes road safety, protects lives, and has the potential to be integrated into modern vehicles for a safer driving experience.

# Key Features
* Breath Analysis: Real-time monitoring of alcohol levels directly from the driver's breath.
* Smart Engine Lock: Automatically disables the vehicle ignition if the threshold is exceeded.
* Instant Alert System: Triggers a warning buzzer and locks the engine when alcohol is detected.
* Proven Performance: Evaluated through physical experiments demonstrating high accuracy and effectiveness.

# Hardware & Components Used
* Microcontroller: Arduino Uno
* Sensors: MQ-3 Alcohol Detection Sensor
* Alert Mechanism: Piezo Buzzer and engine lock
